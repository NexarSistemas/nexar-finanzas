# AI Development Workflow

Proceso recomendado para desarrollar usando IA.

1 analizar código existente
2 identificar componente a modificar
3 diseñar cambio
4 implementar
5 probar localmente
6 actualizar documentación

## reglas

no romper compatibilidad con base de datos

si se modifica una tabla:
crear migración segura

mantener simplicidad
