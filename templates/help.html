Manual de usuario

Clasificación de categorías: Necesario / Prescindible

En Categorías (menú lateral), cada categoría de gastos ahora muestra si es Necesario o Prescindible. Con el botón Marcar prescindible / Marcar necesario podés cambiarla con un clic. Al crear una nueva categoría también elegís su tipo.

  • Necesario: gastos que no podés evitar — alquiler, servicios, salud, alimentación, transporte, educación.
  • Prescindible: gastos opcionales — entretenimiento, suscripciones, ropa, tecnología, salidas.
Análisis en Reportes → Mensual

La sección Reportes (modo mensual) incluye ahora un panel completo de análisis que muestra:

  • Monto y % de gastos necesarios vs prescindibles.
  • Variación respecto al mes anterior (↑ sube, ↓ baja).
  • Detalle por categoría con barra de proporción.
  • Recomendaciones automáticas según tus ratios (sin IA).
  • Botón Analizar con IA para recomendaciones personalizadas con lenguaje natural (requiere API key configurada).
Cierre completo de la aplicación

Al presionar Cerrar app → Sí, cerrar desde el menú superior:

  • La pestaña del navegador se cierra automáticamente (si fue abierta por el script).
  • En Linux/Mac: se envía SIGHUP a la terminal que inició la app, cerrándola también.
  • En Windows: se termina la ventana CMD/PowerShell que ejecutó la app.
Si abriste el navegador manualmente (no por el script), la pestaña mostrará un mensaje indicando que podés cerrarla manualmente.

Clasificación automática de gastos

Al ingresar un nuevo movimiento, escribí la descripción (ej: "Carrefour", "Netflix", "YPF") y la IA sugerirá automáticamente la categoría más adecuada después de un breve momento. Podés Aplicar la sugerencia con un clic o ignorarla. Si la IA no tiene confianza suficiente, no muestra nada.

Asistente financiero en lenguaje natural

El ícono en la esquina inferior derecha abre el chat del asistente. Podés preguntarle en español:

  • ¿Cómo cerré el mes?
  • ¿En qué gasté más este mes?
  • ¿Estoy dentro de mis presupuestos?
  • ¿Cuánto ahorré?
  • ¿Cómo va mi cartera de inversiones?

El asistente tiene acceso de lectura a tus transacciones, cuentas, presupuestos e inversiones reales. Responde en lenguaje natural con datos concretos. Requiere conexión a internet.

Configuración

Antes de usar la IA, configurá tu clave de API en Configuración → Inteligencia Artificial. La clave se obtiene gratis en console.anthropic.com y se guarda de forma segura en tu base de datos local.

Tu clave de API nunca sale de tu equipo — solo se usa para llamar directamente a la API de Anthropic.

Corrección: sección de actualización duplicada

Se eliminó la sección duplicada de Actualización del sistema que aparecía dos veces en Configuración. Ahora solo aparece una vez, usando la versión mejorada con copia de seguridad automática previa.

Corrección: error "falta UPDATE_META.json"

Se corrigió el error que impedía instalar actualizaciones desde el formulario de Configuración. El sistema ahora reconoce correctamente el archivo de manifiesto update_manifest.json incluido en los paquetes de actualización oficiales.

Si al subir el archivo update_vX.X.X.zip recibías el error "falta UPDATE_META.json", ya está resuelto.

Actualización sin pérdida de datos

Desde Configuración → Actualización del sistema podés aplicar nuevas versiones subiendo un archivo .zip oficial. El proceso actualiza solo los archivos del sistema y nunca modifica tu base de datos, historial de transacciones, cuentas, configuración ni copias de seguridad.

Pasos para actualizar:

  1. Descargá o recibí el archivo update_vX.X.X.zip
  2. Ingresá a Configuración en la barra lateral
  3. En la sección Actualización del sistema, seleccioná el archivo
  4. Hacé clic en Aplicar actualización
  5. Reiniciá la aplicación para que los cambios tomen efecto
⚠️ Se recomienda realizar una copia de seguridad desde Configuración antes de actualizar.
Instaladores para Windows y Linux

A partir de esta versión, Finanzas del Hogar está disponible como instalador nativo:

  • Windows (x64 y x86): instalador .exe que incluye Python, todas las dependencias, acceso directo en el escritorio y menú de inicio. No requiere instalar nada por separado.
  • Ubuntu / Debian (Linux): paquete .deb que instala todo automáticamente, crea un lanzador en el menú de aplicaciones y un ícono en el escritorio.
  • Versión portable: ZIP simple que funciona en cualquier sistema con Python 3.10+ instalado.

Corrección: categorías duplicadas

Se corrigió un error que causaba que las categorías predeterminadas se insertaran más de una vez en la base de datos cada vez que se iniciaba la aplicación. La tabla ahora tiene una restricción que impide duplicados, y la migración automática limpia los duplicados existentes al actualizar.

Pantalla de licencia rediseñada

La sección Activar sistema ahora muestra la información de la licencia de forma visual según el tipo:

  • Licencia mensual: muestra la fecha exacta de vencimiento con una alerta para renovar antes de que expire.
  • Licencia permanente o de cliente: muestra el símbolo ∞ y confirma que no hay fecha de vencimiento.

También se mejoró la pantalla de activación en modo DEMO: ahora incluye barras de progreso de uso, los tipos de licencia disponibles y los beneficios de la versión completa.

Cotizaciones de mercado para inversiones

Las posiciones abiertas ahora muestran el precio actual de mercado obtenido automáticamente desde fuentes gratuitas:

  • Acciones argentinas y CEDEARs → Yahoo Finance (ticker con sufijo .BA)
  • Acciones internacionales (USA) → Yahoo Finance (sin sufijo)
  • Bonos y obligaciones → BYMA Open Data
  • FCI (Fondos Comunes de Inversión) → API CAFCI oficial
  • Criptomonedas → CoinGecko
Cálculo automático de ganancias y pérdidas

Por cada posición abierta el sistema calcula:

  • Costo promedio: precio ponderado de todas las compras.
  • Valor a mercado: cantidad × precio actual.
  • Ganancia / Pérdida: diferencia entre valor de mercado y costo total.
  • Rendimiento %: ganancia expresada en porcentaje sobre el capital.
  • Variación del día: cambio porcentual en las últimas 24hs.

Botón de cierre en inicio de sesión

El botón "Cerrar aplicación" también está disponible en la pantalla de inicio de sesión.

Requisitos
  • Python 3.10 o superior
  • Sistema operativo: Windows o Linux
  • Conexión a internet solo para cotización USD (opcional)
Pasos de instalación
  1. Descomprimí la carpeta finanzas_app/ en tu equipo.
  2. Abrí una terminal en esa carpeta.
  3. Instalá las dependencias: pip install -r requirements.txt
  4. Ejecutá la app: python app.py
  5. Abrí tu navegador en: http://127.0.0.1:5000
  6. En la primera ejecución, creá tu usuario administrador.
En Windows (doble clic)

Podés crear un archivo iniciar.bat con el contenido:

@echo off
cd /d "%~dp0"
python app.py
pause

Desde el menú Cuentas podés agregar:

  • Cuenta bancaria: con CBU y alias (Ej: Cuenta Nación, Galicia)
  • Billetera virtual: con CVU y alias (Ej: Mercado Pago, Ualá)
  • Efectivo: para el dinero en mano

Al crear una cuenta, ingresá el saldo inicial. El saldo actual se calculará automáticamente conforme registrés ingresos, gastos y transferencias.

Transferencias entre cuentas

Usá el botón Transferir para mover dinero entre tus propias cuentas sin que afecte el balance general.

DEMO: Máximo 4 cuentas bancarias y 4 billeteras virtuales.

Registrar un gasto
  1. Clic en Nuevo gasto en el menú o dashboard.
  2. Elegí el tipo: Gasto o Ingreso.
  3. Completá: monto, moneda (ARS/USD), fecha, categoría, método de pago y cuenta.
  4. Agregá una descripción opcional y guardá.
Métodos de pago disponibles
  • 💵 Efectivo
  • 💳 Débito
  • 💳 Crédito
  • 🔄 Transferencia bancaria
  • 📱 Billetera virtual (QR, Mercado Pago, etc.)
Categorías

Podés agregar tus propias categorías desde el menú Categorías.

En Presupuestos podés fijar un límite mensual de gasto por categoría.

El sistema muestra un semáforo automático:

  • 🟢 Normal: gastaste menos del 80% del presupuesto
  • 🟡 Advertencia: entre 80% y 99%
  • 🔴 Excedido: superaste el 100%

Las alertas también aparecen en el Panel principal.

Podés definir presupuestos distintos para cada mes y año.

En Reportes encontrás tres vistas:

  • Mensual: ingresos, gastos, balance y gráfico de torta por categoría.
  • Anual: gráfico de barras mes a mes comparando ingresos vs gastos.
  • Semanal: movimientos de los últimos 7 días.
Exportación CSV

Desde la vista de reportes podés descargar tus transacciones en formato CSV para abrirlo en Excel u otra planilla de cálculo.

Gráficos

Copia manual del archivo

Todos los datos están en un único archivo: database.db en la carpeta de la aplicación.

Para hacer una copia: copiá ese archivo a otro lugar (pendrive, nube, etc.).

# Windows (cmd)
copy database.db copia_finanzas_2026.db

# Linux / Mac
cp database.db copia_finanzas_2026.db
Restaurar una copia

Para restaurar: cerrá la app y reemplazá el archivo database.db con tu copia de seguridad.

⚠️ Hacé copias regularmente. La recuperación de datos eliminados no es posible una vez borrados.

La sección Cotizaciones muestra valores actualizados de múltiples fuentes gratuitas sin necesidad de registro ni clave de acceso.

Sin conexión a internet

Actualizar

Presioná Actualizar ahora para consultar valores frescos. Si alguna fuente falla, las demás continúan funcionando.

Desde Configuración → Copias de seguridad podés programar copias automáticas de tu base de datos.

Frecuencias disponibles
  • Diaria: al abrir la app si pasó al menos 1 día.
  • Semanal: cada 7 días al iniciar (opción predeterminada).
  • Mensual: cada 30 días al iniciar.
  • Nunca: sólo copias manuales.
Copia inmediata

Usá el botón Hacer copia de seguridad ahora. El archivo se guarda en backups/ con nombre tipo backup_20260215_143022.db.

Descargar y restaurar

Podés descargar cualquier copia desde la lista en Configuración. Para restaurar, reemplazá database.db con el archivo descargado.

⚠️ Las copias más antiguas se eliminan automáticamente según el límite configurado.
