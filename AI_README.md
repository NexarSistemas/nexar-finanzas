---

# Descripción del proyecto

Nexar Finanzas es una aplicación local para gestión financiera personal.

Permite:

- registrar ingresos y gastos
- gestionar cuentas financieras
- crear presupuestos
- registrar inversiones
- consultar cotizaciones
- generar reportes financieros

La aplicación funciona completamente **offline** y guarda los datos en **SQLite**.

Puede ejecutarse:

- como servidor Flask local
- en una ventana nativa usando **pywebview**
- en navegador si pywebview no está disponible
